# springboot-vue-blog

#### 介绍
SpringBoot+MyabtisPlus+VUE整合的一个博客小项目，比较适合新手，大神勿喷。

> 项目暂时还未完全写完，所以还缺了很多东西

#### 软件架构
后端：SpringBoot+MyBatisPlus+Lombok+HuTools

数据库：MySql 5.7

数据库连接池：Druid

前端：VUE+Element-ui+axios
